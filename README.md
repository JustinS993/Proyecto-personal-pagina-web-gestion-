# Gestión de Citas — Psicología

Sistema de gestión de citas para consultorio de psicología (login, citas, pacientes, usuarios).

## Estructura del proyecto

```
├── css/           → Estilos (colores, layout, componentes)
├── js/            → Lógica de la aplicación
├── index.html     → Página principal
├── iniciar.bat    → Ejecutar la app (doble clic)
├── package.json   → Configuración y scripts
└── README.md      → Este archivo
```

## Cómo ejecutar

**Opción 1:** Doble clic en `iniciar.bat`

**Opción 2:** En terminal: `npm.cmd run dev`

**Usuarios:** Admin: `admin` / `admin123` — Empleado: `empleado` / `empleado123`
